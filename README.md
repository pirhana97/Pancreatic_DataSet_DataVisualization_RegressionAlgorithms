# Pancreatic_DataSet_DataVisualization_RegressionAlgorithms

The dataset is taken from Kaggle's dataset "Urinary biomarkers for pancreatic cancer". The data Visualization and the Regression algotihms applied was done on Google Colab notebook.
The following have been done to the dataset:
1. Exploratory Data Analysis
2. Data Preprocessing
3. Data Cleanup
4. Apply 2 Regression Algorithms i.e. Logistic Regression and Decision Tree Regression
